AI Sign Language Interpreter

Python, Google’s MediaPipe, Machine Learning

ISS Tracker

Python, Linux, Flask

This project used Flask, API, and Docker to build a real-time web app that tracks the International Space Station. It retrieves key metrics, such as speed, location, and altitude, and converts coordinates into readable addresses. The app includes RESTful endpoints, error handling, and unit tests using pytest to ensure accuracy and reliability. Docker was used to containerize and deploy the app across environments.

Dynamic Memory Allocator

C, Python, Perl, Shell

This C programming project optimizes memory management by redefining malloc() and free() functions, introducing a linked list to efficiently manage free memory blocks, and implementing algorithms for improved allocation. It notably increased performance, averaging 1415.79 memory operations per millisecond and enhancing the system's efficiency and responsiveness by 20% compared to the standard approach.
