Python & QuantConnect

AI Sign Language Interpreter

Python & Google’s MediaPipe

This project harnessed the capabilities of OpenCV and Google's MediaPipe to enable real-time, precise tracking of key points on the human hand, creating a groundbreaking sign language recognition system. By analyzing hand formations, it accurately translated sign language gestures into readable text, providing a powerful tool for bridging communication gaps for the deaf and hard of hearing.

International Space Station Tracker

Python, Flask, & Docker

This project is a Flask-based API designed for tracking the International Space Station (ISS). It provides endpoints to access various data about the ISS, including its state vector data, instantaneous speed at specific epochs, nearest epoch data, and location information. Leveraging Flask for the web server, along with libraries like requests and xmltodict, it offers a user-friendly interface for retrieving and analyzing ISS-related data.

Dynamic Memory Allocator

C, Python, Perl, Shell

This C programming project optimizes memory management by redefining malloc() and free() functions, introducing a linked list to efficiently manage free memory blocks, and implementing algorithms for improved allocation. It notably increased performance, averaging 1415.79 memory operations per millisecond and enhancing the system's efficiency and responsiveness by 20% compared to the standard approach.
